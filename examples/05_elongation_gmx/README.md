# Example 03: Uniaxial elongation (GROMACS deform)

This example performs uniaxial deformation with `deform` in GROMACS and extracts a stress–strain curve.

## Prerequisites

Put your input files in this folder:

- `system.gro`
- `system.top`

## Run

```bash
python run_elong.py
```

Outputs will be written to `work_elong/`.
